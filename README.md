# aspnet-core-mvc-with-ef-core

Following the Microsoft tutorial [ASP.NET Core MVC with EF Core - tutorial series](https://docs.microsoft.com/en-us/aspnet/core/data/ef-mvc/?view=aspnetcore-5.0).

The completed solution by Microsoft is in [Github](https://github.com/dotnet/AspNetCore.Docs/tree/main/aspnetcore/data/ef-mvc/intro/samples/cu-final).

## Clean Architecture

Updated the architecture of the solution to match the guidelines in the following articles:

- [Common web application architectures | Microsoft Docs](https://docs.microsoft.com/en-us/dotnet/architecture/modern-web-apps-azure/common-web-application-architectures#clean-architecture)
- [Clean Architecture Solution Template](https://github.com/ardalis/cleanarchitecture). This GitHub repository is referenced from the above article.
- [Why you need Use Cases/Interactors | by Denis Brandi | ProAndroidDev](https://proandroiddev.com/why-you-need-use-cases-interactors-142e8a6fe576)
- [The Clean Architecture | The Clean Code Blog](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html)
